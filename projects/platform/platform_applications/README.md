# Silicon Labs Platform Applications #

Silicon Labs have provided numerous examples of [Platform Applications Example](https://github.com/SiliconLabsSoftware/platform_applications).

We invite the community to share their own projects, share innovative ideas, and collaborate with others. By participating, you help expand the collection of resources available to all developers and foster a vibrant ecosystem around Silicon Labs platforms. Whether you are submitting a new listing, improving documentation, or providing feedback, your contributions are highly valued and appreciated.

## Requirements ##

1. Silicon Labs EFR32 Development Kit
2. Simplicity Studio 5
3. Compatible SDK version, you can install it via Simplicity Studio or download it from our GitHub [gecko_sdk](https://github.com/SiliconLabs/gecko_sdk) or [simplicity_sdk](https://github.com/SiliconLabs/simplicity_sdk)

## How to contribute ##

This repository accepts link submissions (not project source code). To share your project, add a new row to the table below with:

- a link to your public GitHub repository
- a 1-2 sentence description
- a link to your GitHub profile
- append your new row to the bottom of the table (do not reorder or edit existing rows)

See [How to Contribute](../../../README.md#how-to-contribute) and [CONTRIBUTING.md](../../../.github/CONTRIBUTING.md) for details.
